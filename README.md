## Instalação

Use o terminal para instalar:

* Node
```bash
  npm i node
```
* Json Server
```bash
  npm install json-server
```
* Live Server
```bash
  npm i live-server
``` 

## Iniciando o json-server

```bash
  npx json-server --watch db.json
```

## Inicie o live-server

```bash	
  Alt+L+O
```
